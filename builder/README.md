# builder

English Wiktionary 덤프를 영영사전 SQLite 로 변환하는 스크립트.

## 입력
- English Wiktionary 덤프([dumps.wikimedia.org](https://dumps.wikimedia.org)), 또는
  이미 파싱된 [kaikki.org](https://kaikki.org) 의 JSON(wiktextract)

## 출력
- `open-en-dict.sqlite` — 테이블 `entries(word, pos, definition, ipa, example)`
